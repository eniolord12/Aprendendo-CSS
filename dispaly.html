Funções de text

text-alin:

text-alin: left; - Alinha o texto à esquerda

text-alin: right; - Alinha o texto à direita

text-alin: center; - Centraliza o texto

text-alin: justify; - Texto justificado

text-decoration

text-decoration: none; - Remove todas as decorações

text-decoration: underline; - Adiciona um sublinhado

text-decoration: line-through; - Adiciona uma linha através do texto

text-decoration: overline; - Adiciona uma linha acima do texto

text-transform

text-transform: none; - Sem transformação

text-transform: capitalize; - Converte a primeira letra de cada palavra em maiúscula

text-transform: uppercase; - Converte todo o texto para maiúsculas

text-transform: lowercase; - Converte todo o texto para minúsculas

letter-spacing

letter-spacing: px; - Controla o espaçamento entre os caracteres

letter-spacing: normal; - Espaçamento padrão entre os caracteres

word-spacing

word-spacing: px; - Controla o espaçamento entre as palavras em uma linha de texto

word-spacing: normal; - Espaçamento padrão entre palavras

line-height

line-height: px; - Define a altura da linha, ou seja, o espaço entre as linhas de texto dentro de um elemento.

text-indent

text-indent: px; - Define o tamanho do recuo da primeira linha

white-space

white-space: normal; - O texto quebra nas palavras e espaços múltiplos são combinados em um

white-space: nowrap; - O texto não quebra, e é exibido em uma linha contínua

white-space: pre; - Preserva espaços em branco e quebras de linha como estão no código fonte

white-space: pre-wrap; - Preserva espaços e quebras, mas também permite a quebra automática de linha

text-shadow

text-shadow: none; - Remove qualquer sombra de texto

text-shadow: horizontal-offset; - Deslocamento horizontal

text-shadow: vertical-offset; - Deslocamento vertical

text-shadow: blur-radius; - Raio de desfoque

text-shadow: color; - Cor

font- size

font- size: px; - Define o tamanho do texto

font-weight

font-weight: normal; - Espessura normal

font-weight: bold; - Texto em negrito

font-weight: lighter; - Letra fina

font-weight: bolder; - Texto negrito mais espesso

font-weight: 100 à 900; - Define a espessura específica
